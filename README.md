# Lookup Challenge

A simple spreadsheet simulation to motivate the use of lookup functions.

## Requirements

* [Node](https://nodejs.org) > 7.6

## Usage

To start the development server

```sh
npm start
```

To build for production

```sh
npm run build
```

To preview the production build
```sh
npm run preview
```
